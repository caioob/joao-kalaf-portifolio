- **jsdom test document has no head meta** ‚Äî the title/description effect had nothing to update. Added a `<meta name=description>` to `src/test/setup.js` mirroring index.html, so the effect (and its assertion) behave as in the browser.

## 5. Decisions of record

- **OG image as SVG-source ‚Üí generated PNG via a committed `og:image` script**, not a build-time image library ‚Äî keeps zero runtime/heavy-dev deps, makes the asset reproducible after a brand/accent change (token philosophy), and ships the PNG crawlers require. `rsvg-convert` rendered a clean bold sans that reads as on-brand.
- **Rubik kept as the display face but flagged pending approval** (design ¬ß3). Open item, _not_ resolved here: `Rubik-Regular.ttf` is a single static weight, so `font-semibold`/bold display text renders synthetic-bold ‚Äî revisit (variable Rubik or weight set, ideally woff2) when the font is approved.
